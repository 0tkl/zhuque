# 朱雀仿宋

![](preview.png)

「朱雀仿宋」是[璇璣造字](http://trionestype.com/)的開源仿宋字體計畫，志在最終提供高品質、支援多語言的正文字型解決方案。

仿宋作為「宋、黑、仿、楷」中文四大基礎字體之一，長期缺乏在開源授權下的選擇。我們希望「朱雀仿宋」能彌補這項空缺，為開源軟體與 TeX 社群貢獻一份力量。

上世紀初，仿宋活字一度品類繁盛、應用廣泛；然而隨時間推移，市面上的仿宋形態逐漸收斂為「華豐真宋」系統，這也是今日多數人所熟悉的樣貌。「朱雀仿宋」改刻自民國仿宋活字「南宋」，在滿足現代排印需求的前提下，嘗試為正文字型帶來一縷新鮮氣息。

目前發佈的「朱雀仿宋」仍屬「預覽測試版」，仍處於專案早期階段；無論設計風格、字數或字形與排版品質，都不代表未來正式版的最終狀態。

## 設計故事

開發一套開源仿宋字型的想法始於 2021 年初，是繼「未來熒黑」專案之後，我們認為更具意義且更具挑戰性的任務。隨著研究與技術的演進，特別是近年來 AIGC 技術的快速發展，實現此構想已變得可行。

「朱雀仿宋」改刻自民國活字「南宋」，試圖為這套沉睡數十年的字體帶來新生。「南宋」於 1932 年由「百宋鑄字印刷局」發行，韓佑之為設計原稿，鄒根培雕刻字模。問世後廣泛應用於書籍與期刊排印。「南宋」的字形設定與多數正文字體不同：重心稍偏右下；字形中宮收緊、撇捺舒展；筆畫的粗細對比強烈；兼具含蓄秀美與力量感，於標題與正文排印皆具書卷氣。

在「南宋」的基礎上，「朱雀仿宋」適度降低筆畫對比、規整字形結構與版面效果，同時保留「南宋」的獨特韻味與活字趣味。

「朱雀」為中國傳統文化中的「四象」之一，為南方七宿之神；其名稱與「南宋」歷史淵源以及「璇璣造字」品牌相呼應。
